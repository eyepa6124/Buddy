## ✨ What Buddy does

Buddy is an AI text assistant for Android. It helps you rewrite text inside any app without switching screens.

Type a trigger at the end of your text, like:

- `/fix`
- `/formal`
- `/reply`
- your own custom trigger

Buddy then rewrites the text using Gemini or any OpenAI-compatible provider.

## 📲 Set up Buddy on Android

After you install Buddy, open it on your Android phone and finish the setup.

### 1. Grant access

Buddy needs accessibility access to work in other apps. This lets it watch the text you type and replace it with the version you want.

To turn it on:

1. Open Android settings
2. Go to Accessibility
3. Find Buddy in the list
4. Turn the service on
5. Confirm the prompt

### 2. Choose your AI provider

Buddy can work with:

- Gemini
- OpenAI-compatible providers
- Other API-based text models

Add your provider key and endpoint in the app settings if needed. If you use Gemini, enter the Gemini key. If you use another service, paste the API details the app asks for.

### 3. Pick your trigger words

Buddy watches for a trigger at the end of your text. You can use built-in triggers or create your own.

Common examples:

- `/fix` for grammar cleanup
- `/formal` for a more polished tone
- `/reply` for a quick response
- `/short` for a shorter version
- `/custom` for your own style rule

### 4. Test it in any app

Open a text field in an app like:

- Messages
- WhatsApp
- Email
- Notes
- Social apps

Type your text, add a trigger, then send it through Buddy. The app rewrites the text using the style you chose.

## 🧭 How to use Buddy

Buddy keeps the process simple.

1. Type your message in any app
2. Add a trigger at the end
3. Let Buddy rewrite the text
4. Copy the result back into the app or replace the current text if supported

Example:

- Before: I need this done asap /formal
- After: I would appreciate your help with this as soon as possible

Another example:

- Before: can you send me the file /reply
- After: Could you send me the file?

## ⚙️ Tips for better results

Use short, clear prompts. Buddy works best when the text is easy to understand.

Try this:

- Keep the main idea in one sentence
- Add the trigger at the end
- Use one request at a time
- Pick the right tone for the app you are using

Good examples:

- “Please review the document /formal”
- “Send a polite follow-up /reply”
- “Make this shorter /short”
- “Fix the grammar and keep the meaning /fix”

If the result looks wrong, try a shorter prompt or a different trigger.
